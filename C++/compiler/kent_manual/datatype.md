# 数据类型

## byte 
字节类型，语法示例：
```k
byte a = NULL // 初始化为空（数值为 0）
a = 0b00100000 // 二进制赋值
a = 0x12 // 十六进制赋值
a = 12 // 十进制赋值
a = 'a' // 字符 ASCII 码赋值

## int
整数类型，语法示例
```k
int a = NULL // 初始化为空(数值为0)
a = 0b00000000,00000000,00000000,01010101 // 二进制赋值
a = 0b0,0,0,01010101 // 二进制赋值,缩写
a = 12 //十进制赋值
